Files included:

- services.html

- contact.html
- assets/logo.png (your uploaded logo)
- assets/favicon.png (same logo, recommended to replace with .ico if desired)
- assets/style.css
- assets/script.js

How to publish on GitHub Pages:
1. Create a new public GitHub repo (e.g. 'haulhedge').
2. Upload all files and the 'assets' folder to the repository root.
3. Commit changes.
4. Go to Settings → Pages → Branch: select 'main', folder '/(root)', Save.
5. Visit https://YOUR-USERNAME.github.io/REPO-NAME/

Replace Formspree endpoint in contact.html with your Formspree form ID or your backend endpoint to receive messages.
